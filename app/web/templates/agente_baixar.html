Ainda não tenho o agente instalado — me mostra como instalar
Installer `.exe` ainda em prep (Fase 9.5). Por enquanto, rode em modo dev (Python) seguindo os passos abaixo. Em poucos dias liberamos o AchadinhosAgent.exe com auto-update.

Setup dev (3 minutos)

  1. 1️⃣

    Instale Python 3.11+ e Chrome

    python.org · google.com/chrome

  2. 2️⃣

    Clone o repo e instale dependências

    git clone https://github.com/silviosvargas/achadinhosv3.git
cd achadinhosv3/agente
python -m venv .venv
.venv\Scripts\activate
pip install -e .
  3. 3️⃣

    Rode o agente (sem token)

    python -m agent.main --sem-tray

    O agente vai subir em modo "aguardando pareamento" (ainda sem token). Daí volta nesta página e clica em "Conectar meu agente" acima — o pareamento é automático, sem precisar copiar/colar token.

  4. 4️⃣

    Faça login inicial nas plataformas (1× cada)

    python -m agent.login_ml         # escaneia ou loga no ML
python -m agent.login_whatsapp   # escaneia QR do WhatsApp Web

    Sessões ficam salvas no perfil persistente. Renove só quando expirar (~30 dias).

Detalhes técnicos

  • Roda em Windows / Mac / Linux (testado em Windows 11 / Python 3.13).
  • Comunicação cifrada via WebSocket (em produção, wss://).
  • Pareamento via HTTP local em 127.0.0.1:5577 (ADR-009).
  • Token JWT de 1 ano, gravado em %APPDATA%\Achadinhos\config.json.
  • Pode revogar o agente a qualquer momento em Agentes.
  • Múltiplos agentes por usuário (PC casa + PC trabalho) — cada um com seu token.
