Instalar o agente Achadinhos

O agente roda no seu PC em background. Ele faz as buscas no Mercado Livre (via Selenium) e posta no WhatsApp Web pra você. Sem ele, só Telegram funciona (via cloud).

Status: instalador .exe ainda em prep — por enquanto, rode o agente em modo dev (Python). Em poucos dias liberamos o AchadinhosAgent.exe com auto-update.

Setup dev (3 minutos)

  1. 1️⃣

    Instale Python 3.11+ e Chrome

    python.org · google.com/chrome

  2. 2️⃣

    Clone o repo do agente e instale dependências

    git clone <url-do-repo-achadinhos-agent>
cd achadinhos-agent
python -m venv .venv
.venv\Scripts\activate
pip install -e .
  3. 3️⃣

    Rode o setup (vai pedir email + senha)

    python -m agent.setup

    O setup vai logar nesta conta, criar um agente automaticamente, e gravar o token em %APPDATA%\Achadinhos\config.json. Você não precisa copiar/colar token nenhum.

  4. 4️⃣

    Faça o login inicial nas plataformas (1× cada)

    python -m agent.login_ml         # escaneia ou loga no ML
python -m agent.login_whatsapp   # escaneia QR do WhatsApp Web

    Sessões ficam salvas no perfil persistente. Renove só quando expirar (~30 dias).

  5. 5️⃣

    Rode o agente

    python -m agent.main --sem-tray

    Quando ver ws.conectado, está pronto. Volte aqui no dashboard, clique em Lote ou em qualquer busca e clique "▶ Rodar".

Detalhes técnicos

  • Roda em Windows / Mac / Linux (testado em Windows 11 / Python 3.13).
  • Comunicação cifrada via WebSocket (em produção, wss://).
  • Token JWT de 1 ano, gravado em arquivo do usuário (não em variável de ambiente).
  • Pode revogar o agente a qualquer momento em Agentes.
  • Múltiplos agentes por usuário (PC casa + PC trabalho) — cada um com seu token.
