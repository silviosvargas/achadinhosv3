Como funciona a tag de afiliado?

Cada marketplace tem um sistema próprio de afiliação:

  • Mercado Livre: tag aparece como ?matt_word=<sua_tag> na URL.
  • Shopee: URL encurtada via s.shopee.com.br/<id>.
  • Amazon: ?tag=<sua_tag> (formato nome-20).
  • Magalu / AliExpress / TikTok: também por query param ou shortlink.

O sistema aplica sua tag automaticamente em toda postagem de produto dessa plataforma. Se você não cadastrar uma tag, a tag do admin Achadinhos é usada (fallback).
